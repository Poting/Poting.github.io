SMTP

在 ASP .NET Core 1.x 時並沒有包含SMTP，所以就需要安裝一些套件來達成寄信的功能，像是
MailKit、Mailgun等等之類的，到了2.0時包進了 SMTP，這真的hen棒!! (可以直接copy code過來修改)。

好的，所以什麼是SMTP?

簡單郵件傳輸協定 (Simple Mail Transfer Protocol) 是事實上的在Internet傳輸email的標準

根據WIKI 的定義非常的淺顯易懂，就是一個處理郵件的協定，電腦上安裝了 SMTP 之後就可以寄信拉，通常都是透過port 25來建立連線，正常來說應該不會有程式占用 25 port 不過保險起見，還是確認一下比較好。

SMTP是一個 “推送” 的協定，如果要接收信件，就需要 POP3、或IMAP來實作，
但是這不是這次的重點，所以改天再來說吧。
